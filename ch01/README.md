### 1. 강의노트에서 몰랐던 용어에 대하여 자세히 조사하시오 <br>
   제네릭함수: 제네릭 함수는 제네릭 형식 매개 변수를 사용하여 선언된 함수이며, 호출 시 형식 매개 변수 대신 실제 형식이 사용된다.<br>
### 2. 알고리즘 설계와 코딩 중 무엇이 중요하고 그 이유는 무엇인가?<br>
   알고리즘 설계가 잘 되어야 코딩을 할 수 있기 때문에 알고리즘 설계가 무척 중요하다.<br>
### 3. 클래스와 객체가 무엇인지 설명하시오.<br>
   클래스는 객체를 만드는 틀이며 사용자 정의 자료형이고 객체는 그런 틀 안에서 생겨난 실체로 클래스형 변수이다.<br>
### 4. 절차지향 프로그래밍 언어와 객체지향 프로그래밍 언어의 차이를 조사하라.<br>
   절차지향은 프로그램을 실행 순서에 따라 파악하여 순서처리에 집중하는 방식으로 직관적이며 실행속도가 빠르다는 장점이 있다. 대표적인 언어로 c언어가 있다. 객체지향은 프로그램을 클래스와 객체를 중심으로 구성하며 보안성, 재사용성이 높다는 장점이 있다. 대표적인 언어로 c++, java 등이 있다. <br>
### 5. 라이브러리가 무엇인지 자세히 설명하시오.<br>
   프로그램을 만들 때 많이 사용되는 기능을 미리 작성해 놓은 코드를 모아둔 것으로, 개발자가 기능을 쉽게 사용할 수 있도록 제공된다.<br>
### 6. 라이브러리가 목적파일 형태로 존재하는 이유를 설명하라.<br>
   컴파일 시간 단축이나 편의성을 위함도 있지만, 목적파일로 변환된 소스파일을 링크단계에서 실행파일로 만들기 위해 연결되기 때문이다.<br>
### 7. 링크과정이 필요한 이유를 설명하라.<br>
   목적파일과 라이브러리를 연결하여 정상적인 실행파일을 만들기 위해 필요하다.<br>
