# 실습과제 1
### 객체의 생성과 소멸의 의미를 설명하라
객체 생성은 운영체제가 관리하는 메모리 공간을 프로그램에서 사용할 수 있게 됨을 의미하고
객체 소멸은 사용하던 메모리 공간을 운영체제에 반납하여 더 이상 그 공간을 사용할 수 없게 됨을 의미한다.

### 지역변수(지역객체), 전역변수(전역객체)의 차이를 조사하라, 선언위치, 접근범위, 생성시점 초기화 여부 등의 관점에서 비교하라.
지역변수는 함수나 블록 내부에서 선언되며 해당 블록 내에서만 접근이 가능하다. 변수는 함수 호출 시가 아니라 블록이 실행될 때 생성되며 자동으로 초기화되지 않기 때문에 초기값은 쓰레기값을 가진다. 또한 스택 영역에 저장되며 변수 오염을 방지할 수 있다.

전역변수는 함수 외부에서 선언되며 프로그램 전체에서 접근할 수 있다. 프로그램이 시작될 때 생성되고 자동으로 0으로 초기화된다. 데이터 영역에 저장되며 함수 간 데이터 공유가 편리하지만 변수 충돌이나 디버깅이 어려울 수 있다.

### 프로그램 실행 후 main함수가 실행되기 전에 어떤 일이 처리되는지 조사하라
프로그램이 메모리에 로드되고 실행 준비 과정이 이루어진다. 이 과정에서 전역변수와 정적변수가 초기화되며 런타임 환경이 설정되고 전역객체가 생성된다.

### main함수가 종료 후 프로그램이 종료하기 전에 어떤 일이 처리되는지 조사하라 
main 함수가 종료된 이후에는 정리 작업이 이루어진다. 전역변수와 정적변수가 소멸되고 전역객체의 소멸이 발생하며 사용하던 자원이 운영체제에 반환된다
