# 실습문제 2
### 매크로상수 NULL이 무엇인지 조사하라, C/C++언어에 서정의의차이,정의된헤더파일,사용용도등
| 표준   | 기호     | 정의                          | 헤더파일     | 자료형        |
|--------|----------|-------------------------------|--------------|---------------|
| C언어  | NULL     | #define NULL ((void *)0)      | <stddef.h>   | void*         |
| C++98  | NULL     | #define NULL 0                | <cstddef>    | int           |
| C++11  | nullptr  | 키워드임                      | 컴파일러내장 | std::nullptr_t |

### 널문자와 NULL의 차이를 설명하시오, 정의, 자료형, 메모리크기, 실제값, 사용용도 등
널문자('\0')는 문자형(char) 데이터로서 값이 0인 문자를 의미하며 문자열의 끝을 표시하기 위해 사용되는 문자이고, NULL은 값이 0인 포인터 상수로서 어떤 유효한 주소도 가리키지 않음을 나타내며 포인터를 초기화하거나 더 이상 참조할 대상이 없을 때 사용하는 값이다.
### 요즘은 NULL 대신에 nullptr을 사용하는데 nullptr은 무엇인지조사하시오
nullptr은 C++11에서 새롭게 도입된 포인터 전용 null 상수로, 기존의 NULL이 정수 0으로 처리되어 함수 오버로딩 시 모호성이 발생하는 문제를 해결하기 위해 도입되었으며, std::nullptr_t 타입을 가지는 값으로서 오직 포인터가 아무것도 가리키지 않음을 명확하고 안전하게 표현하는 것이다.
